# Network Programming Project

Проект по сетевому программированию на Python с эхо-серверами.

## Решенные проблемы в улучшенных версиях:

1. ✅ **Отключение клиента по команде** - команды `exit` и `quit` теперь работают
2. ✅ **Обработка отключений клиентов** - сервер не падает при разрыве соединения
3. ✅ **Пустой ввод** - при нажатии Enter без текста выводится подсказка
4. ✅ **Выключение сервера** - команда `stop server` корректно выключает сервер
5. ✅ **Корректное завершение** - клиент завершается с сообщением "Process finished with exit code 0"

## Как использовать:

### 1. Простой сервер (1 запрос - 1 ответ):
```
cd echo_01_simple
python echo_01_server.py
python echo_01_client.py
```
### 2. Расширенный сервер (много запросов, 1 клиент):
```
cd echo_02_advance
python echo_server_02_server.py
python echo_server_02_client_01.py
```
### 3. Многопоточный сервер (много клиентов):
```
cd echo_03_mult
python echo_server_03_server.py
python echo_server_02_client_01.py
```

## Команды для клиентов:

- `exit` или `quit` - отключиться от сервера
- `stop server` - выключить сервер
- Любой другой текст - будет возвращен в верхнем регистре
- Пустой ввод (Enter) - получите подсказку

## Особенности:

1. **Расширенный сервер** (`echo_02_advance/echo_server_02_server.py`):
   - Обрабатывает одного клиента за раз
   - Поддерживает команды отключения
   - Корректно завершает работу при `stop server`

2. **Многопоточный сервер** (`echo_03_mult/echo_server_03_server.py`):
   - Одновременно работает с несколькими клиентами
   - Использует потоки для параллельной обработки
   - Глобальный флаг для корректного выключения
   - Уведомляет клиентов при выключении сервера
